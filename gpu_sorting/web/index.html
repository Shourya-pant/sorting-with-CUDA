About ParallelSort

ParallelSort is a powerful sorting visualization tool that implements various sorting algorithms in both JavaScript and simulated CUDA.

The application demonstrates how parallel sorting algorithms can significantly outperform traditional sorting methods for large datasets.

Version: 1.1.0

New Features:

  • GPU acceleration for better performance
  • Support for multiple sorting algorithms
  • Interactive performance visualization
  • Improved responsiveness and animations

How to Use ParallelSort

Getting Started:

  1. Select a sorting algorithm from the dropdown menu
  2. Choose the dataset size to sort
  3. Select execution mode (GPU or CPU)
  4. Click "Start Sorting" to begin the visualization

Understanding the Results:

After sorting completes, you'll see:

  • A visualization of the sorted data
  • Performance metrics comparing CPU vs. GPU execution times
  • Speed improvement calculation

Supported Algorithms:

  • Bitonic Sort: Efficient parallel sorting algorithm, ideal for GPU execution
  • Merge Sort: Divide-and-conquer algorithm with good parallelization potential
  • Quick Sort: Fast comparison sort with limited parallelization capability
